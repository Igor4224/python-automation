# conversor_xml

Desenvolvi uma solução de automação em Python para processamento de Notas Fiscais Eletrônicas (NF-e), capaz de realizar a leitura de múltiplos arquivos XML, extrair informações relevantes e consolidar os dados automaticamente em uma planilha Excel.
O projeto foi desenvolvido com foco na redução de atividades manuais, aumento da produtividade e padronização do tratamento de documentos fiscais.

Funcionalidades implementadas:
• Leitura automatizada de arquivos XML de NF-e;
• Identificação de diferentes estruturas de documentos fiscais;
• Extração de informações como número da nota, empresa emissora, cliente, endereço de entrega e peso da carga;
• Tratamento de campos opcionais e validação de dados;
• Consolidação das informações em DataFrame utilizando Pandas;
• Exportação automática para planilhas Excel.

Tecnologias utilizadas:
• Python
• xmltodict
• Pandas
• OpenPyXL (via exportação Excel)

Benefícios da solução:
• Redução do tempo de processamento de documentos fiscais;
• Eliminação de tarefas repetitivas;
• Maior confiabilidade na consolidação de dados;
• Facilidade para análises e relatórios operacionais.
Este projeto reforçou minha experiência em automação de processos, manipulação de dados estruturados e desenvolvimento de soluções voltadas para eficiência operacional.
